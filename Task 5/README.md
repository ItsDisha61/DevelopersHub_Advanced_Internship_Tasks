# Task 5: Interactive Business Dashboard in Streamlit

## Task Objective

The objective of this project is to develop an interactive business dashboard using the Global Superstore dataset. The dashboard helps analyze sales, profit, and customer performance through interactive filters and visualizations, enabling better business decision-making.

---

## Your Approach

- Loaded the Global Superstore dataset.
- Cleaned the dataset by removing missing values and duplicate records.
- Built an interactive Streamlit dashboard.
- Added filters for Region, Category, and Sub-Category.
- Displayed Key Performance Indicators (KPIs):
  - Total Sales
  - Total Profit
  - Top Customer
- Created interactive charts for:
  - Sales by Region
  - Profit by Region
  - Sales by Category
  - Top 5 Customers by Sales
- Displayed the filtered dataset for detailed analysis.

---

## Results and Findings

- Successfully developed an interactive business dashboard using Streamlit.
- Users can dynamically filter data by Region, Category, and Sub-Category.
- The dashboard provides quick insights into Total Sales, Total Profit, and Top Customer.
- Interactive charts help identify business performance across different regions and product categories.
- The dashboard improves data visualization and supports data-driven business decisions.
