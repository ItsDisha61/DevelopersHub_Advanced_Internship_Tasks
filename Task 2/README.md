# Task 2: Customer Segmentation Using Unsupervised Learning

## Objective

The objective of this project is to segment customers based on their spending behavior using K-Means Clustering and provide suitable marketing strategies for each customer group.

## Approach

- Loaded and explored the Mall Customers dataset.
- Performed Exploratory Data Analysis (EDA).
- Encoded categorical features and applied feature scaling.
- Used the Elbow Method to determine the optimal number of clusters.
- Applied the K-Means clustering algorithm.
- Used PCA to visualize customer segments.
- Proposed marketing strategies for each identified customer segment.

## Results and Findings

- Customers were successfully segmented into different clusters based on their characteristics.
- PCA provided a clear visualization of the customer groups.
- The identified segments can help businesses design targeted marketing campaigns.
- Customer segmentation supports better decision-making and improves marketing effectiveness.
