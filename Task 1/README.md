# Task 1: Term Deposit Subscription Prediction

## Objective

Predict whether a bank customer will subscribe to a term deposit based on a marketing campaign using machine learning classification models.

## Approach

- Loaded and explored the Bank Marketing dataset.
- Encoded categorical features.
- Trained Logistic Regression and Random Forest models.
- Evaluated the models using Confusion Matrix, F1-Score, and ROC Curve.
- Used SHAP to explain model predictions.

## Results and Findings

- Both models successfully predicted customer subscription.
- Random Forest achieved better overall performance.
- SHAP identified the most important features influencing customer subscription.
- The project demonstrates how Explainable AI can improve customer behavior analysis.
